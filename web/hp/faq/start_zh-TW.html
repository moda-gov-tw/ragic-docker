如何刪除表單或是一個上面的頁籤？

將滑鼠移到功能表中您想要刪除的表單或頁籤上，右方將會出現一小箭頭圖示，點選箭頭，再選擇「刪除」即可；或者直接在您想要刪除的表單或頁籤上，點選滑鼠右鍵，接著選擇「刪除」亦可。

如果您不小心刪除了表單或頁籤想要復原，您可以與我們聯絡support@ragic.com，我們將立即幫您處理。

怎麼調整儲存格的高度或是寬度呢？

要修改儲存格的高度與寬度，您可以先進入設計模式(點選表單上「修改設計」)，接著就像使用Excel一樣，將滑鼠移到最上方A、B、C…每欄的邊界處，鼠標將變為雙箭頭，此時即可利用拖拉方式，設定儲存格的寬度；同樣的，移到最左方1、2、3…欄的邊界處，即可以調整儲存格的高度。

調整完畢後記得「儲存」，之後每個人所看到的這張表單，都將會是您調整過後的樣子。

您也可以在一般瀏覽時，利用上述方式調整儲存格的高度及寬度，以便觀看，但這些調整在下次進來這張表單時不會被記住。

能夠將不同表單的欄位的資料互相做連結嗎？

當然可以，這是Ragic最大的特色!

將不同表單的欄位互相連結，僅需要2步, 我們以Sales表單及Customer表單為例:

1. Linked Fields: 假設我們在Sales表單新增一Customer欄位，提供使用者在新增訂單時選擇客戶之用，Customer欄位的資料來源為Customer表單. 我們稱此Customer欄位為linked field.

你會發現Customer欄位的資料內容為Customer Id，即Customer表單中的第一個欄位，在Ragic，表單的第一個欄位我們稱做 Key Field, 在設計模式時為該欄位為深藍色。 Linked Fields會關聯到另一個表單的Key Field, 就像Sales表單的Customer欄位關聯到Customer表單的Customer Id。

2. Loaded Fields: 當使用者在Sales表單中的Customer欄位選擇了一個客戶，Ragic可以連帶地將他在Customer表單中的電話、住址等資料帶入Sales表單中。這可以大大減少資料輸入的時間，讓你的系統看起來更聰明，我們稱這些自動帶入的欄位為loaded fields.

Linked Field及Loaded Fields的設定很簡單，使用我們的連結管理便可輕鬆完成(在設計模式，點選「連結其它表單」按鈕，即可進入連結管理)。

在連結管理中，您必須先設定Linked Field，首先，在左邊畫面(即您正在設計的表單)，點選您既有的欄位(Ex.Customer) 或任何一個空白儲存格，接著，系統便會自動將它與右方表單中的Key field相連結(如同我們之前所說的，Linked Field必定與Key Field相連)。

接著您可以設定Loaded Fields，設定的方式與剛剛一樣，在左方點選您既有的欄位或任何一個空白儲存格，接著點選右方您希望自動帶入的欄位。

當左方為空白儲存格時，連結管理員會自動幫您新增該欄位。

如何匯入手上Excel或是CSV的資料呢？

Ragic有很方便的資料匯入精靈，可以幫助使用者將既有的Excel或CSV資料匯入Ragic資料庫。如果您已經設計好表單，想匯入資料，您可點選表單上方「工具」按鈕，選擇「匯入\從Excel匯入資料」，接著依照匯入精靈的引導，一步步設定即可完成資料匯入。

如果您尚未設計表單，您也可以直接使用Excel或CSV來建立表單；您可點選點選表單上方「工具」按鈕，選擇「匯入\用Excel建立新表單」，一樣依照匯入精靈的引導，一步步設定即可完成表單的建立同時匯入資料。

如何把表單嵌入到我的網站或部落格？

要將Ragic表單嵌入到網站或部落格非常簡單，點選表單上的「工具」按鈕(如下圖)，選擇「嵌入這張表單」，接著會出現一訊息方塊，裡面含有一段HTML嵌入語法；將訊息方塊中的嵌人語法複製下來，到您的網店或部落格貼上，其它訪客便能使用您在Ragic建立的表單囉!

您也可以利用訊息方塊下方的選項，進一步設定表單在您的網站或部落格中要顯示的樣子。

可以上傳檔案，圖片，或是影片到Ragic上嗎？

可以，所有的檔案類型都可以上傳到您的Ragic。設計時，只要將您的欄位「輸入型態」選為檔案上傳或圖片上傳；或者從左方「新增元件」頁籤中，拖拉一個「檔案上傳」或「圖片上傳」元件到您的表單中，即可擁有檔案上傳/圖片上傳功能。

使用時，使用者只要點擊該欄位，便會出現檔案上傳視窗供使用者上傳檔案或圖片。資料儲存後，如為檔案，該欄位會以超連結呈現，當使用者點選超連結，即可下載該檔案；如為圖片，則直接顯示於儲存格中，而當使用者點選圖片時，即可下載該圖片的全幅檔案。

我可以把資料存成Excel嗎？

可以，只要點選「工具」按鈕，選擇「下載\下載成Excel檔案」就可以囉。Ragic同時也提供「友善列印」功能，一樣在「工具」，選擇「下載\友善列印」即可。

不論在表單頁還是列表頁，您都可以將表單內容存成Excel；在表單頁，存的是一筆資料，而在列表頁，則是該表單所有的資料哦!

我可以自行開發HTML頁面來將資料寫入Ragic表單中嗎？

如果您會開發HTML表單，那麼要寫入Ragic表單非常簡單，我們來看一個例子，假設您要新增資料到「範例：竉物店」中的merchandise表單：

1. 找出您要透過HTML表單來儲存的欄位，並記下它們的欄位編號，欄位編號位於設計模式中，欄位名稱的下方。

2. 建立您的HTML表單如下，在HTML表單中每個欄位的name需使用上面所說的欄位編號，它會以參數方式傳入Ragic，Ragic藉由欄位編號得知HTML欄位與Ragic欄位的對應關係；表單送出時，送出的Action為Ragic表單的API網址(即該Ragic表單的網址，惟將www.ragic.com 換成 api.ragic.com).

  <!-- Your HTML form saves data to the same form URL, except change www.ragic.com to api.ragic.com -->
  <!-- Adding the web parameter means that api server returns html instead of default JSON -->
  <form action="http://api.ragic.com/start/petstore/1?html" method="POST">

    <!-- Put down the field id as the parameter name for each field to map them to a field on the Ragic form -->
    <p>Item Id: <input type="text" name="1000001" /></p>
    <p>Item Name: <input type="text" name="1000003" /></p>
    <p>Item Price: <input type="text" name="1000005" /></p>

    <!-- Selections work the same way, just pass the selection value as parameter value -->
    <p>Item Category:
      <input type="radio" name="1000002" value="Dog" /> Dog
      <input type="radio" name="1000002" value="Cat" /> Cat
    </p>
  </form>

3. 確認Ragic表單的權限設定，使用者仍需擁有Ragic表單存取權限，才能從HTML表單將資料存入Ragic，如果您的表單是開放大眾使用的，我們建議您可將該表單的所有人群組設為佈告欄式使用者.

我要怎麼備份我的資料呢？

Ragic servers每天都會自動進行備份，如果您有特殊的需求，Ragic也提供各用戶手動備份的功能，您可從「功能捷徑」選擇資料，進入資料庫相關頁面。

接著選擇下載帳號資料備份，下載備份檔。

備份檔為資料庫的壓縮備份，僅為資料庫還原所使用，如果您需要可閱讀的資料，您可參考8. 我可以把資料存成Excel嗎？，在列表頁利用「下載\下載成Excel檔案」或「下載\下載成純文字檔案」功能，將Ragic資料庫中的資料備份出來。

Ragic有沒有智慧型手機的app？

Ragic有一個Android app，您可以到這邊下載，或是直接在Google Play上搜尋 "Ragic"。我們的iOS iPhone/iPad app即將上線，上線前您可以直接用手機上的瀏覽器登入 www.ragic.com 也可以直接瀏覽編輯資料。

如何讓別人也看到表單上的資料？

很簡單，只要提供您表單的網址，即可讓其它人看到這張表單的資料，Ragic每一個帳號、每一張表單甚至每一筆資料，都有一個獨立的網址，您可在瀏覽器的網址列取得它.

不過，由於不是每個人都有Ragic帳號，如果您要提供沒有帳號的人也能查看，您必須先修改該表單的權限(在設計模式中，點選「表單設定」頁籤，接著點選「存取權限」頁籤)。

在這裡，您可以看到您的帳號下所有的群組，在這張表單的權限為何，在此Ragic提供了一個特別的群組「所有人」，意即不論是否為Ragic的使用者，是否為您的使用者均屬此群組。

您可以為每個群組設定其在這張表單的存取角色為何，如果有使用者分屬於2個群組，則他的權限會以2個群組中存取權限較高的為主。

每次權限設定後，記得「儲存」您的設計，以便生效；列表頁與表單頁的存取權限是相同的，您只需要設定其中之一即可。

權限設定後，如有人仍無法存取表單，可請他登出後重新登入，讓權限重新生效。

同時兩個人更新同一筆資料會怎麼樣？

當使用者開始編輯資料時，該筆資料便會被Ragic鎖定，如同時間有另一位使用者要編輯同一筆資料時，Ragic會提示該使用者該筆資料正在編輯中，該使用者應等到前一位使用者儲存或取消，才能開始編輯，以避免資料互相覆蓋。

但在某些特殊情況下(如聯絡不到使用中的人又急須修正資料..)，Ragic也允許使用者手動取消鎖定，唯必須注意的是，取消鎖定後，該筆資料將會以最後編輯儲存的內容為主，原使用者所輸入到一半的資料，有可能有資料覆蓋的風險。

要怎麼把我同事加到資料庫裡面呢？

點選功能表中的「設定」頁籤，選擇「使用者管理」，您將進入到使用者管理頁面，您會看到您帳號下所有的使用者，在此點選「+新增」，接著填寫該使用者資料即可。

儲存後，Ragic會自動發送一封Eamil (內含暫時密碼)給該使用者，以便該使用者第一次登入Ragic。登入後，該使用者可自行俢改密碼。

是否能夠只有某些人看得到某些欄位？

可以，我們可以利用「複製工作表」來達成。有些時候，相同的表單，可能因使用者角色的不同，所能看到的欄位也不同，利用「複製工作表」，我們可以快速複製一張表單的設計，再增刪部份欄位即可。

將滑鼠移到功能表中您想要複製的表單上，點選滑鼠右鍵，選擇「複製工作表」(或者點選右方小箭頭，再選擇「複製工作表」)，Ragic便可複製產生另一個相同的表單，兩張表單將存取同一份資料庫中的資料，不論您在哪一張表單中輸入資料，二張表單都能看到您所新增的資料；同樣的，當您在其中一張表單刪除某一筆資料，另一張表單中也不會再看到該筆資料。

表單複製後，我們接著便可為這二張表單設計不同的存取權限，Ex.第一張表單僅能給A群組存取，第二張表單則給B群組使用。

最後您可以設計這二張表單，新增或移除些欄位，讓它們擁有不同的顯示內容了。

可以設定每個人能看得到哪幾筆資料嗎？

可以，每張表單都能設定每個群組所扮演的角色為何，每位使用者依照他所屬的群組，會在不同的表單中取得不同的角色，當使用者角色為問卷式使用者，他只能存取自己所新增的資料；而當使用者角色為佈告欄式使用者，則他可以看到所有的資料，但是只能編輯自己新增的資料。

一般而言，使用者只能存取自己所新增的資料，但有些設計可能會有例外。當表單中有選擇使用者或選擇群組欄位時，如又勾選了選定使用者有資料權限，則屆時，該筆資料的擁有者將會變成該欄位所指定的人或該群組下所有的人。

我有很多廠商或是客戶的外人偶爾會使用我們的資料庫，有辦法買少一點使用者嗎？

有幾個方法能夠幫助您處理大量使用者的情況

1. 公開表單: 您可以設計您的表單為公開表單，即任何人不需登入Ragic即可使用。如果您的系統不需要特別的權限，您可將您的表單存取權限設定為所有人都是問卷式使用者，那麼每個人都將能填寫表單；如果設成佈告欄使用者，那麼每個人不僅能填寫表單，還能看到其他人所填寫的內容。

如果您需要在資料輸入後，隨時了解該筆資料的最新情況，那麼您可以考慮接下來的做法：E-mail認證。

2. E-mail 認證: Ragic有一個很簡單的方法，能夠提供非Ragic的使用者，暫時擁有Ragic的存取權限。您可在表單中設計一Email欄位，在「欄位設定」頁籤中的「其它」頁籤，勾選其Email認證屬性，之後每當表單存檔時，系統便會自動發送Email至該欄位所填寫的Email位址，Email中會提供本表單連結，讓收件者可利用該連結登入Ragic.

透過Email認證，Ragic可以知道登入者是誰(不論是否擁有Ragic帳號)，可以判斷登入者的存取權限。本功能通常是為了讓使用者了解他所新增的資料，是否有變更。

這種方法的缺點是 (1) 這些使用者總是得透過Email來連結Ragic (2) 這些使用者無法以群組加以管理其在Ragic的使用權限。

3. 大量授權: 雖然 Ragic的收費並不昂貴，但當您需要大量帳號時，加起來也不便宜，因此Ragic另有優惠的大量授權方案，以及私有主機方案，提供選擇。您可以聯絡 sales@ragic.com，告訴我們您的需求，我們將幫助您選擇最適合的授權方案。

簡易版跟專業版差別在哪呢？

1. 簡易版有5張表單的限制而專業版則沒有限制，表單指的是您所設計的使用者輸入介面，而非資料，假設Customer表單中有100筆客戶資料，這樣只算是1張表單。

2. 簡易版不支援server-side Javascript 工作流程，在專業版中，所有的表單都能透過server-side Javascript 工作流程撰寫程式，來處理複雜的商業邏輯。基本上，任何Ragic Builder現行功能無法滿足的需求，您都可以透過撰寫server-side Javascript 工作流程來解決。

要如何開始正式付款使用或是升級我的Ragic？

點選Ragic右上方「升級正式版!」，選擇您要升級的Ragic版本。

您會被導引至加密的線上刷卡頁面，您的Ragic帳號將會在付款後自動升級

您也可以隨時從 [開始使用Ragic] => [Ragic帳號設定] => [帳號付款]，來進行升級 / 購買。

正式使用後，免費試用的天數會不見嗎？

不用擔心，試用期不會因為您升級正式版而消失哦，正式版的起算時間，會在試用期滿之後才開始計算。

儲存空間用完了會怎麼樣？

Ragic的上限多半是「軟性上限」，也就是說當您的應用超過上限時，並不會馬上停止運作。我們會e-mail通知您需要升級您的帳戶以滿足現階段的使用量。
